map (c++ STL)

Semon Liu Lv3
  2024-04-21 2024-04-21 Created   2024-04-21 23:06:47 2024-04-21 23:06:47 Updated  
  • c++ 
 

std::map 是 C++ 标准模板库（STL）中一个非常实用的容器，它存储键值对（key-value pairs）并基于键（key）自动排序。在内部，std::map 通常使用红黑树实现，这为大多数操作（如插入、删除、查找）提供了对数时间复杂度的性能。

基本特性

  • 唯一的键：每个键在 map 中必须是唯一的。
  • 自动排序：元素根据键自动排序，排序规则默认是 std::less<Key>，这意味着键会以升序排列。
  • 直接访问：提供通过键直接访问对应值的能力，如果键不存在，会自动创建一个与该键关联的值。

主要成员函数

  1. 插入和删除
    • insert(std::pair<const Key, Value>(key, value))：向 map 中插入键值对。如果键已存在，则插入不会发生。
    • erase(const Key& key)：从 map 中删除键及其关联的值。
    • clear()：移除所有键值对，清空 map。
  2. 查找和访问
    • find(const Key& key)：返回一个迭代器到给定键，如果键不存在，则返回 end()。
    • count(const Key& key)：返回特定键的元素数量（对于 map 而言，结果为 0 或 1）。
    • operator[]：通过键访问值。如果键不存在，将会创建一个新的元素并插入到 map 中。
  3. 大小和容量
    • size()：返回 map 中键值对的数量。
    • empty()：如果 map 为空，则返回 true。
  4. 迭代器
    • begin() 和 end()：提供遍历 map 的方法。由于 map 是有序的，迭代器会按照键的排序顺序访问每个键值对。
