set(c++ STL)

Semon Liu Lv2
  2024-04-21 2024-04-21 Created   2024-04-21 23:04:10 2024-04-21 23:04:10 Updated  
  • c++ 
 

一段描述性的话语 std::set 是 C++ 标准模板库（STL）的一部分，它提供了一个存储唯一元素的集合，这些元素按照特定的排序准则进行排序。set 在内部通常实现为红黑树，这意味着它可以在对数时间内完成大多数关键操作，如插入、删除和搜索。

基本特性

  • 唯一性：set 中的元素必须是唯一的。
  • 有序性：元素根据其值自动排序，默认使用 std::less<T>，这意味着元素将以升序方式存储。
  • 自动排序：当元素被插入到 set 中时，它会自动被插入到正确的位置以保持集合的有序性。

主要成员函数

  1. 插入和删除
    • insert(const T& value)：向集合中插入值，如果值已存在，则操作不会改变集合。
    • erase(const T& value)：从集合中删除值。
    • clear()：移除所有元素，清空集合。
  2. 查找和访问
    • find(const T& value)：返回一个指向找到的值的迭代器，如果未找到则返回 end()。
    • count(const T& value)：返回特定值的元素数量（对于 set 而言，结果为 0 或 1）。
    • lower_bound(const T& value) 和 upper_bound(const T& value)：返回一个迭代器，分别指向不小于（大于等于）和大于特定值的第一个元素。
  3. 大小和容量
    • size()：返回 set 中元素的数量。
    • empty()：如果 set 为空，则返回 true。
  4. 迭代器
    • begin() 和 end()：提供了遍历集合的方式。由于 set 是有序的，迭代器将按照元素的排序顺序遍历元素。
